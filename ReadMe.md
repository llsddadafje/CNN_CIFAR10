# 使用CNN进行CIFAR10分类


## 目录

### dataset:  CIFAR10数据集，打开项目后请自行创建文件夹并放入cifar10数据
### logs:    用于tensorboard画图日志
### Adagrad.py：  使用Adagrad优化器运行
### Adam.py：  使用Adam优化器运行
### SGD.py：  使用SGD优化器运行
## 环境
### python3.7，pytorch1.12.0 其他安装对应版本即可
### 在logs的同级目录下使用命令行  ternsorboard --logdir=logs   运行tensorboard
